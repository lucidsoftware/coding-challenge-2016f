COMPETE                                NERDS

PROGRAM                                VICTORY

Summary

        Given a grid of uppercase letters and a list of words, solve the word search and output the same grid, with all the letters not found in one of the words replaced with a lowercase x. Words may appear left-to-right, top-to-bottom, up-diagonal, or down-diagonal.

Description

        The first line of input will consist of two integers, r and c, the number of rows and columns in a word search puzzle, respectively. The next r lines of input will each contain c space-separated characters, which may be any of the 26 uppercase English letters (A-Z). The next line will contain a single integer n, which is the number of words to find in the puzzle. The following n lines will each contain a string wi. Each character in wi will only contain uppercase English letters (A-Z).

        Each string wi will be formed in the word search puzzle exactly once. There are four different ways words may be formed in the puzzle:

  • Left to right: Every character appears one column to the right of the preceding character.
  • Top to bottom: Every character appears one row below the preceding character.
  • Up diagonal: Every character appears one row above and one character to the right of the preceding character
  • Down diagonal: Every character appears one row below and one character to the right of the preceding character

        A character is considered used in the word search puzzle solution if it is present in forming at least one string wi. All characters that do not contribute to forming at least one string wi are not considered used in the word search puzzle solution.

        Your program will produce r lines of output, each containing c space separated characters. For each corresponding character in the input, output the same character if it is used in the word search puzzle solution, or a lowercase ‘x’ character if it is not used.

Additional Constraints

  • No word will have a length greater than the larger of r and c
  • No word will have a length less than 3
