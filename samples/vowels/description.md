# Vowels

## Description

Count the number of vowels (a,e,i,o,u,y) in a “word”. Include both uppercase and lowercase vowels in your count.

## Input

A sequence of alphabetic characters (no spaces), with length at most 10,000.

## Output

The number of vowels.
