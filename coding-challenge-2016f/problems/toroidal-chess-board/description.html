King's Warpped Journey

Description

Imagine a chess board wrapped around a Torus as shown in the diagram above. This wrapping will cause what were the top and bottom edges of the chess board to be connected, as well as what were the left and right edges of the chess board to be connected. The diagram shows how the coordinate system works across the toroidal and poloidal axes when the chess board is wrapped around the torus.
You will be given the starting position of a King, and a destination where the King must move. In chess a King may move to any directly adjacent or directly diagonal square in a single turn. Determine the minimum number of turns required for a King to traverse from the given starting and destination coordinates, allowing for the King to move across what would be the "edges" of a 2-d chess board.

Input

The first line is a number N. N testcases follow. Each test case consists of two coordinates formatted as X Y. Each coordinate will appear on a separate line of input.

Output

Output a single integer for each test case indicating the minimum number of turns that are required for the king to move from the given starting coordinates to the given destination coordinates.

Examples

Input Input 
1
0 0
0 1

2
0 0
1 2
0 0
