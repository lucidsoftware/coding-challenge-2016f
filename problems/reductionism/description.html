Reductionism

Description

As a programmer, we like to break big problems down into smaller problems. To prepare for the task of defining several acronyms, your task is to print each word with one letter per line.

Input

The input will be one word, containing only lowercase (a-z).

Output

Print the word ready to be reduced, with one character on each line.
